# @synra/capacitor-preferences

Lightweight Capacitor key-value preferences (`get` / `set` / `remove`) for web, iOS, Android, and Electron (via `@synra/capacitor-electron` bridge).

## Install

To use npm

```bash
npm install @synra/capacitor-preferences
```

To use yarn

```bash
yarn add @synra/capacitor-preferences
```

Sync native files

```bash
npx cap sync
```
