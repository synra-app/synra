# @synra/capacitor-device-connection

Capacitor plugin for device session connection and messaging.

## Install

To use npm

```bash
npm install @synra/capacitor-device-connection
```

To use yarn

```bash
yarn add @synra/capacitor-device-connection
```

Sync native files

```bash
npx cap sync
```
